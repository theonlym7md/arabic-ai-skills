## 🚀 One-Command Installation

Add the skill directly to your Claude Code, Cursor, Antigravity, or Agent CLI with a single command:

```bash
npx skills add https://github.com/theonlym7md/arabic-ai-skills --skill arabic-uiux-master
```

Or install the advanced reasoning engine:
```bash
npx skills add https://github.com/theonlym7md/arabic-ai-skills --skill arabic-intelligence
```

## ⚡ 60-Second Overview

Raw LLMs struggle with Arabic UX writing. They default to literal translation, administrative clichés (*"في إطار حرصنا"*, *"يرجى التكرم"*), and un-localized phrasing.

**Arabic AI Skills** provides an exhaustive 10,000+ word Arabic UI/UX Masterclass, Microcopy Lexicon, AI Slop Anti-Cliché Bans, RTL Typography guidelines, and a 10-Stage Guided Reasoning Engine.

## 💥 The Problem vs 🎯 The Solution

| Challenge | Raw LLM Generation (AI Slop) | Arabic AI Skills Output |
| :--- | :--- | :--- |
| **GovTech Fine Notice** | *"في إطار حرصنا المستمر يرجى التكرم بالعلم بأنه لا يوجد لديكم أي مخالفات..."* ❌ | **"سجلك خالي من المخالفات"** <br>*"لا توجد أي مخالفات مسجلة بحقك حالياً."* ✅ |
| **FinTech Card Retry** | *"عذراً حدث خطأ غير متوقع يرجى المحاولة لاحقاً أو الاتصال بالدعم"* ❌ | **"تعذر استكمال عملية الدفع"** <br>*"يرجى التأكد من توفر الرصيد أو استخدام بطاقة أخرى."* ✅ |
| **Cognitive Load** | High (25+ words of administrative fluff) | Low (Direct, active, human-grade brevity) |
| **Cultural Localization** | Translated English syntax | Authentic Saudi & Gulf product phrasing |
