# Brand Engine: Disruptor & Media (لغة الشباب والإعلام الإبداعي)

## Context
Activated for modern content platforms, media startups, and gaming (e.g., Thmanyah, Telfaz11, B22 Games). 

## Core Philosophy: "Cringe-Free Authenticity"
The language must bridge the gap between Modern Standard Arabic (MSA) and Conversational Saudi/Gulf Arabic ("العامية البيضاء"). It avoids trying too hard to sound young (which leads to "Cringe"). It speaks to the user peer-to-peer.

## Tone of Voice Parameters
*   **Focus:** Engagement, Curiosity, Storytelling, Authenticity.
*   **Emotion:** Friendly, Direct, Witty, Relevant.
*   **Style:** Functional Writing with visual chunking. Short, punchy, active sentences.

## The "Living Fusha" Index (مؤشر الفصحى الحية)
Use only Arabic vocabulary that a university student would use in a serious conversation in a coffee shop. Do NOT use archaic dictionary words.

## Vocabulary Matrix
| Avoid (Corporate/Boring) | Use (Conversational/Punchy) | Rationale |
| :--- | :--- | :--- |
| المقالات الصحفية | صحافة تصنع الفارق | Sell the promise, not just the category. |
| تسجيل الدخول للحساب | ادخل حسابك | Drop the formal nouns, use direct conversational imperatives. |
| المزيد من المعلومات | السالفة وما فيها / اقرأ القصة | Pique curiosity (Information Gap theory). |

## Rules of Engagement
1. **The Density Rule:** Delete 30% of the words in your first draft. If a word does not add new meaning or emotion, kill it.
2. **The 7-Second Rule:** The Hero section headline MUST convey the core promise in under 7 seconds without needing the sub-description.
3. **No Passive Voice:** Start with a strong active verb.
4. **Peer-to-Peer:** Do not act like a corporate entity giving orders. Act like a friend sharing a great discovery.
