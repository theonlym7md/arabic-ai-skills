# 🤖 تعليمات وكلاء الذكاء الاصطناعي (Global Agent Rules)

هذا الملف مخصص لوكلاء الذكاء الاصطناعي (AI Agents) الذين يقومون بتعديل أو إضافة مهارات جديدة في هذا المستودع. 
**أيها الوكيل الذكي، اقرأ هذه القواعد جيداً قبل أي تعديل!**

## 1. الهيكلية الإلزامية (Mandatory Structure)
- لا تضع أي مهارة في الجذر.
- كل مهارة جديدة يجب أن توضع في مسار `skills/<skill-name>/`.
- كل مهارة يجب أن تحتوي كحد أدنى على: `SKILL.md` (الدستور) و `README.md` (التسويق).

## 2. قواعد جودة المهارات (Skill Quality Standard)
- **لا للـ Slop**: يُمنع منعاً باتاً برمجة الوكيل في `SKILL.md` لاستخدام الترجمة الحرفية أو الكلمات الكسولة مثل "قم بـ" أو "تم بنجاح".
- **آلة الحالة (State Machine)**: كل مهارة تُضاف هنا يجب ألا تكون مجرد Prompt سطحي. يجب أن تقود المستخدم عبر أسئلة (Grill Mode) لجمع السياق قبل التنفيذ.

## 3. التعامل مع الكود
- عند إضافة مهارة جديدة، تأكد من تحديث الجدول في ملف `README.md` الموجود في الجذر (Root).
- احرص على عدم كسر الـ GitHub Actions الموجودة في مجلد `.github`.
